Задание

Необходимо выполнить в отдельном JS файле, подключенному к отдельной HTML странице

  1. Переменная lang может принимать 2 значения: 'ru' 'en'.
    Написать условия при котором в зависимости от значения lang будут выводится дни недели на русском или английском языке. Решите задачу
    • через if
    • через switch-case
    • через многомерный массив без ифов и switch.
  2. У нас есть переменная namePerson.
    Если значение этой переменной “Артем”, то вывести в консоль “директор”, если значение “Максим”, то вывести в консоль “преподаватель”, с любым другим значением - вывести в консоль “студент”.
    Решить задачу с помощью нескольких тернарных операторов, без использования if или switch
  3. Создать свой репозиторий на GitHub и поместить туда папку с усложненным заданием
